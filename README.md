# Faith-Driven Developer Portfolio

A clean, professional portfolio built with React, featuring smooth animations and biblical inspiration.

## Features

- Modern, responsive design that looks great on all devices
- Smooth animations powered by Framer Motion
- Biblical verses and encouraging messaging throughout
- Project showcase section with GitHub integration
- Easy to customize and deploy
- Performance optimized with Vite

## Quick Start

### 1. Install Dependencies
```bash
npm install
```

### 2. Start Development Server
```bash
npm run dev
```

Your portfolio will be running at `http://localhost:5173/`

### 3. Customize Your Portfolio

See the **[CUSTOMIZATION_GUIDE.md](./CUSTOMIZATION_GUIDE.md)** for detailed instructions on:
- Adding your projects
- Uploading images
- Updating contact information
- Customizing colors and text
- Deploying to the web

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build locally

## Technologies Used

- **React** - UI framework
- **Vite** - Build tool and dev server
